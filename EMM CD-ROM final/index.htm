The Eilenberg-MacLane Machine

A program designed to compute the integral (co)homology of Eilenberg-MacLane spaces

 

  1. Introduction

    The Eilenberg-MacLane machine is a program which computes the integral (co)homology groups of 1-connected Eilenberg-MacLane spaces of the form K(Z/2^s,n). This version, written in C++, is based on the results of my PhD thesis (Integral Cohomology of Finite Postnikov Towers, see Appendix B, pp. 87-89).

  2. Installation

    You will need a LaTeX distribution installed and working on your computer, as well as a DVI or a PDF viewer (Acrobat Reader) and some common decompression tools (unzip and/or StuffIt Expander, depending on your platform).

    Packages of the Eilenberg-MacLane Machine are provided for Windows and MacOS. The unzipped folders contain everything you need: the executable file and the LaTeX supertabular package.

  3. Usage

    - run the EMM executable file;
    - a console appears and prompts some values (warning, n has to be greater or equal to 2);
    - the machine outputs the calculations in the LaTeX file named output.tex;
    - compile this file through LaTeX (this requires the supertabular package);
    - the table, which is a DVI or a PDF file, is produced;
